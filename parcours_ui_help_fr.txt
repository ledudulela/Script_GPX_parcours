1. Faire une extraction de la zone utile (exemple: france)
   (astuce: pour obtenir les coordonnées de la zone, utiliser gpsvisualizer.com > draw on map)
   Afficher la configuration: menu App. > Afficher la configuration
   Saisir manuellement les coordonnées de la zone à extraire
   Sauvegarder le fichier de configuration: menu App. > Sauvegarder la configuration
   Lancer l'extraction: menu BDD > Extraire une zone
   Eventuellement renommer le fichier ainsi obtenu: menu Bdd > Enregistrer sous (.dat)
   Sélectionner dans la liste, la base de données à utiliser


2. Afficher la zone utile:
   Aller sur gpsvisualizer.com > make a map
   Téléverser le fichier "base de données" résultant de l'extraction


3. Identifier les points utiles au parcours
   Repérer visuellement les points utiles sur gpsvisualizer.com (noms visibles au survol des points avec la souris)


4. Obtenir les coordonnées des points utiles (Waypoints)
   Reporter le nom des "points utiles" dans l'application et exécuter la recherche pour obtenir leurs coordonnées
   menu Recherche > Rechercher les points


5. Sauvegarder la liste des "points utiles" (Waypoints) dans un fichier .csv
   menu Application > Enregistrer le contenu
   Ces points serviront à l'élaboration du plan de vol.
   Les points peuvent être de type Aéroport, VOR-DME, NDB, ILS, Points fixes
   Exemple: parcours_all_points.csv


6. Créer le parcours:
   6.1.
   Dans le cadre principal, conserver tout ou partie des points utiles (il est possible que tous ne soient pas inclus dans le parcours)
   Sauvegarder la liste des Waypoints dans un fichier .csv (pour une utilisation ultérieure)(*)
   Exemple: parcours_waypoints.csv
   (*) Le fichier de Waypoints, après conversion en GPX, peut être transféré sur l'application SDVFR (accepte uniquement des waypoints).

   6.2.
   Convertir leur type en Trackpoint en cliquant sur la case Trackpoint
   Puis menu Outils > Ajouter Distance & Bearing pour calculer la distance entre chaque point consécutif
   Supprimer les premières lignes désormais inutiles précédant le résultat ainsi obtenu.

   Si une déclinaison magnétique existe:
      Observer les valeurs de bearing de chaque point avant correction et,
      Exécuter le menu Outils > Correction du bearing Géographique / Magnétique 
      Indiquer la déclinaison magnétique puis cliquer sur OK
      Observer les valeurs de bearing de chaque point après correction.
      Supprimer les premières lignes inutiles précédant le résultat ainsi obtenu.

   Sauvegarder votre parcours (Trackpoints avec distances) dans un ou plusieurs fichiers .csv 
   Exemples: 
      parcours_trackpoints.csv
      parcours_trackpoints_sans_declinaison.csv
      parcours_trackpoints_avec_declinaison.csv

   Trackpoints "géographiques" ou Trackpoints "magnétiques" ?
   Le choix se fait en fonction de la finalité.
   S'il s'agit d'afficher les points sur une carte, les Trackpoints avec bearings "géographiques" suffisent.
   S'il s'agit de paramétrer les instruments de bord pour un vol, les Trackpoints "magnétiques" sont nécessaires.
   gpsvisualizer.com affiche les coordonnées, la distance, le bearing des points dans une info-bulle.
   Le choix se fait donc également en fonction de ce que l'on souhaite afficher dans gpsvisualizer.


   6.3.
   Afficher ce parcours (Trackpoints) sur gpsvisualizer.com
   Il est possible d'afficher les points utiles ou tous les points de la zone, et les Trackpoints (parcours) en même temps.


7. Obtenir les coordonnées d'un point par rapport à un point donné, une distance et un bearing
   Il est possible d'intégrer au parcours un point fictif n'existant pas dans la base de données.

   7.1.
   Pour cela, ouvrir le fichier des "points utiles" (Waypoints),
   Supprimer les lignes inutiles et ne conserver que les entêtes et la ligne du point

   7.2.
   Exécuter le menu Outils > Ajouter distance & bearings (pour compléter les colonnes avec ",dist,bearing")
   Supprimer les premières lignes désormais inutiles avant le résultat ainsi obtenu

   7.3.
   Modifier les valeurs de dist et bearing, ces valeurs serviront pour le calcul.
   Exécuter le menu Outils > Coord. à partir de: lat,lon,dist,bearing

   Il est possible d'obtenir plusieurs "points distants" à partir d'un même point de référence.
   Ceci peut être pratique par exemple dans le calcul de radiales d'un arc DME.
   Pour obtenir tous les points distants en une fois,
   Dupliquer autant de fois que nécessaire la ligne de référence (7.3.)
   Modifier les valeurs de dist et bearing de chaque ligne.
   Pour un arc DME, on gardera toujours la même distance mais avec un bearing décaler de 10 ou 20°.
   Exécuter le menu Outils > Coord. à partir de: lat,lon,dist,bearing

   Si besoin, en cas de déclinaison magnétique, utiliser l'outil prévu à cet effet.
   Bonus caché: 
   Lors de la saisie de la valeur de déclinaison, ajouter la lettre T derrière celle-ci.
   Cliquer sur OK
   Le résultat affichera les points avec leur nouveau bearing ET également leur tangente.
   Une tangente est ici composée d'un segment avec un point géographique à chaque extrémité.
